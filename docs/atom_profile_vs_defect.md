# Atom Profile Versus Defect

For nodal Calabi--Yau threefolds, HodgeCY treats classical defect as a coarse
numerical shadow of flexible-sector compression. The classical defect is
important, but by itself it does not record how flexible contributions cluster,
glue, or organize into finer candidate block structure.

The HodgeCY atom profile refines this coarse shadow by recording candidate
block/relation structure. For smoothing-bridge examples, the initial block
profile is induced by the double-line labeling inherited from the source plane
arrangement: under genericity assumptions, each double line contributes four
expected nodes.

This remains a candidate-level profile until two further steps are completed:

- local node verification for the smoothing-bridge singularities;
- classical defect computation for the resulting nodal double octic.

The `84/84a` pair remains a motivational/control pair in this repository. Its
source-arrangement arithmetic distinction does not automatically imply different
smoothed atom spectra, so the comparison layer is designed to keep pending
computations explicit rather than collapsing them into premature claims.
