# Cynk--Kocel--Cynk 2026 source material

This folder is for source material from Cynk--Kocel--Cynk, "Classification of Double Octic Calabi--Yau Threefolds Defined by an Arrangement of Eight Planes II", arXiv:2602.19413.

The paper classifies 455 combinatorial types and gives equations/Magma routines. The current ingestion is additive and does not modify v0.1.0 HodgeCY artifacts.

Any manually copied equations or tables should record provenance. Do not place copyrighted full text into the repository unless reuse is explicitly allowed; keep this folder to small extracted data files, provenance notes, and machine-readable records needed for reproducible computations.
